## PROJECT ONE: Build an LED circuit
Description: Building an LED circuit following the YouTube playlist tutorial from SunFounder Maker Education "Arduino
Tutorial for Beginners 2023"


Components Used: Two 330Ω resistors, Two 100Ω resistors, Breadboard, UNO R3 Controller Board, LED, wires


What I learned: 
  - How to use basic digital output signals for blinking effects
  - Applying basic circuit laws and essential diode knowledge to get the required resistor values and forward voltage drops
