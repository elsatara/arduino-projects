## PROJECT ONE: Build an LED circuit
Description: Building an LED circuit following the YouTube playlist tutorial from SunFounder Maker Education "Arduino
Tutorial for Beginners 2023"


Components Used: Two 330Ω resistors, Two 100Ω resistors, Breadboard, UNO R3 Controller Board, LED, wires. See calc_led2.jpeg for resistor calculations. The chosen resistor values result in ~14mA (not the 20mA maximum rating). Values were chosen based on the components I had available and what is safe for the LED.


What I learned: 
  - How to use basic digital output signals for blinking effects
  - Applying basic circuit laws and essential diode knowledge to get the required resistor values and forward voltage drops


File guide:
1. LED2_circTut.ino: Contains the code I used to implement the LED circuit and blinking effect
2. led2_pjt.jpeg: A photograph of the final LED circuit
3. calc_led2.jpeg: A photograph of the calculations I did to find the right resistor values for the LED ratings I had to fit into.
 Achtung! Achtung! Correction: I made a mistake in the section labelled 3. R is not 140Ohms! The total resistance value is around 215Ohms and current is approximately 14mA. (I included this for anyone who thinks better on paper like I do :) )
